Tags HTML


Entender sobre as Tags HTML é importante, pois elas são marcadores usados ​​para delimitar seções de um documento HTML.

Cada tag HTML tem um significado semântico e define um bloco de conteúdo que pode ser manipulado pelo navegador. Por exemplo, as tags podem delimitar um parágrafo, texto em negrito, texto itálico e assim vai.

O uso das tags HTML são extremamente úteis porque permitem que os navegadores interpretem e apresentem corretamente o conteúdo. Além disso, as tags HTML também podem ser usadas para aplicar estilos a um documento HTML usando folhas de estilo em cascata (CSS).


Tipos de tags

  • Titulos
  • Parágrafos
  • Destaque
  • Elemento de comentário lateral
  • listas
  • Redirecionamento
  • Citação
  • Formatar Texto

Títulos (voltar)

As tags HTML que usamos para criar títulos em páginas web são: h1, h2, h3, h4, h5 e h6. O significado dessas tags é o mesmo, que é definir títulos, mas o nível de importância é diferente para cada uma, onde h1 tem o nível mais alto e h6 o mais baixo.

OBS: as boas práticas recomendam que seja usado apenas um h1 por página

Abaixo mostra como são os títulos

  1. <h1>: Título

  2. <h2>: Título

  3. <h3>: Título

  4. <h4>: Título

  5. <h5>: Título
  6. <h6>: Título

Parágrafo(voltar)

O elemento HTML <p> representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco.


Destaque(voltar)

O Elemento HMTL <mark> representa um trecho de destaque em um texto.

Nesse exemplo foi usando para destacar a palavra. destaque

Além do mark, temos também o <i> que representa o itálico, o <strong> que representa o negrito e o <sub> que representa o sublinhado

Caso você queira corrigir algo de errado que você colocou mas não deseja excluir, você pode utilizar a tag <del> que ela risca o que você deseja: Falei besteira, agora estou falando corretamente.


Elemento de comentário lateral (voltar)

O elemento <small> representa comentários laterais e letras pequenas, como direitos autorais e texto legal, independentemente de sua apresentação estilizada.

você pode identificar o small ao lado de cada título com a palavra voltar

OBS: o redirecionamento que é feito não é devido a tag small, isso veremos mais a frente


Listas(voltar)

Caso você queira listar algo você terá duas opções

  • <ol> = Listas Ordenadas
  • <ul> = Listas Não Ordenadas

Um exemplo de listas ordenada foi a utilzada para listar os tipos de títulos que variam de h1 a h6>

Já um exemplo de lista não ordenada foi utilizado para listar os tipos de listas ol e ul

A tag <li> é usado para representar um item que faz parte de uma lista.
